# Palette (Vue)

This is a port of my original AngularJS repository [Palette](https://github.com/jgefroh/palette) into Vue 3 I wrote as a weekend project in 2016, done with Claude.

Palette is a simple drawing tool to make silly marker drawings or diagrams.

# Installation and running
```
npm install
npm run dev
```
