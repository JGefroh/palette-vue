# Palette

A modern, feature-complete, intuitive drawing application.

Designed for creating quick doodles, markups, wireframes, and diagrams with precision and ease.

![palette](https://github.com/user-attachments/assets/9f20fab8-e2d1-4ee9-92f4-5080685cf617)


## Key Features
* Comprehensive and simple drawing tools — Brush, lines, rectangles, circles
* Advanced color management — Interactive color picker, eyedropper sampling, customizable color themes, and image color detection
* Object manipulation — Full selection, transformation, rotation, and resizing capabilities
* Simple text - font-size, color, style, and alignment support
* Tab support - multiple tabs for multiple images
* Local persistence — Automatically save images locally or download them to your computer
* Image integration — Native drag-and-drop for image import
* Power-user keyboard shortcuts — Complete keyboard support for rapid workflow optimization
* Full undo/redo — Complete operation history support
